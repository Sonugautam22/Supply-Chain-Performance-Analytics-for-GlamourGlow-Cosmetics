# Supply-Chain-Performance-Analytics-for-GlamourGlow-Cosmetics
This project focuses on analyzing and improving the supply chain performance of GlamourGlow Cosmetics using data visualization techniques. A dynamic dashboard was developed in Tableau to track key metrics such as inventory levels, order fulfillment, delivery timelines, and supplier performance.

The dashboard provides clear insights into operational efficiency, helping identify delays, stock issues, and areas for optimization. It enables better decision-making by presenting complex data in a simple and interactive format.

# Tools Used: Tableau, SQL
# Key Focus: Data Analysis, Dashboard Design, Supply Chain Insights
